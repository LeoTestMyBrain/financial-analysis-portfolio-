### 01 · SET50 Bank Stock Screener
**Tools:** Python · yfinance · Pandas · Matplotlib

Automated stock screener for SET50 banking sector using
Justified P/BV valuation model.

**Methodology:**
- Justified P/BV = (ROE − g) / (CoE − g)
- Assumptions: g = 3%, CoE = 10%
- Signal: BUY (>15%), HOLD (0–15%), AVOID (<0%)
